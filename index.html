_Coffea_ is an _Eclipse_ plug-in linking the [_Eclipse_ Java Development Tools](http://www.eclipse.org/jdt/) (JDT) to UML editors. A [previous version](http://coffea.sourceforge.net) hosted on sourceforge used the [_Eclipse_ UML2Tools](http://wiki.eclipse.org/MDT-UML2Tools) editors. According to Ed Merks, these tools are [no longer being maintained](http://www.eclipse.org/forums/index.php?t=msg&goto=544976). Therefore, this _Coffea_ version does not work with _Eclipse_ distributions above _Galileo_. A re-factoring work aims to adapt the initial work for [_Eclipse_ Papyrus](http://www.eclipse.org/papyrus/).
